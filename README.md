# If You Follow
A 2D side-scrolling platformer developed in Unity that explores companion-based gameplay through a trust-driven buddy system.

## Recognition
🏆 Best Game Award – Queen's Computing Showcase

## Overview
If You Follow is a 2D platformer created over a two-month development period for a game design course at Queen's University.
The game focuses on the relationship between the player and a companion character whose behaviour changes based on trust and player interaction. 
Throughout the game, the companion can follow, wait, lead, or lag behind depending on the situation, creating gameplay centered around cooperation and companionship.

## Features
- Companion-based gameplay
- Trust-driven buddy behaviour
- Dynamic follow system
- Hand-holding mechanic
- Scripted obstacle sequences
- 2D platforming movement system
